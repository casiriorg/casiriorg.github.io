## Computación Cuántica Aplicada

[QDR — Qiskit Data Reuploading](../qiskit-data-reuploading/index.md)

Librería Python instalable vía `pip`, compatible con scikit-learn, que implementa clasificadores cuánticos mediante la técnica de *Data Re-Uploading* sobre Qiskit 2.x. Primera implementación de este tipo disponible en el ecosistema Qiskit con API sklearn-compatible, primitivas V2 y soporte para hardware IBM Quantum real.

`python` `qiskit` `quantum-ml` `sklearn`

## Robótica y Sensores

### [RPLiDAR Pointer](https://github.com/casiriorg/s2p-rplidar-pointer-python)

Librería Python para control y adquisición de datos con sensores RPLiDAR S2 y S2E de Slamtec. Proporciona una interfaz de alto nivel para escaneo, filtrado y visualización de nubes de puntos 2D, orientada a aplicaciones de navegación, mapeo y detección de obstáculos en plataformas robóticas y UAS.

`python` `lidar` `robotics` `uas`
